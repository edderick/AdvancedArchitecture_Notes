[edit] Intel Core i7

[edit] QuickPath

Point-to-point processor interconnect

  • Alternative to the front-side-bus
  • All uni-directional
  • Components
    • QPI: Quick path hub
    • Data links: 2 20-lane point-to-point data links, one in each direction with a seperate clock pair in each direction.
      • 4 Control bits
      • 16 Data bits
  • Units of transfer
    • Flit: Basic unit of transfer, 80-bit flit transferred in 2 clock cycles. 64 bits for data, 8 for error detection, 8 for link-layer header

[edit] AMD

[edit] HyperTransport

Bidirectional serial/parallel high-bandwidth, low-latency point-to-point link

  • Usage: To link processors and the bridge together to communicate
  • Features
    • Multi-link high-speed interconnect
      • From 200 MHz to 3.2Ghz
      • Double data rate: Send data on both the rising/falling edges of the clock signal
    • Packet-oriented: 32-bit packets
    • Power-managed
  • Protcol: Transactions and sessions
    • Communication management: Managing communication accross network of opterons
    • Routing
  • Problems
    • Broadcasting: Broadcasting traffic to large numbers of processors - slows everything down and doesn't scale well
    • Differential signalling: one end going down, the other end going up - limits the length and speed
    • Distance restricted: due to differential signalling
    • No Off-system interconnect: Does not permit off-system interconnect

[edit] Horus

  • Solution: Solves the problem of broadcasting traffic to more clusters and allows cache coherency between much larger amounts
  • Can link up to 8 Horus chips
  • How it works
    • Creates a Pseudo-Opteron (the Horus chip), connects to four real Opterons via HyperTransport
    • Handles translation between local and off-bord cache coherency HyperTransport communications
  • Directory: The data being shared is placed in a common directory, maintaining coherence between the caches. Processors ask the directory for permission to load an entry from primary memory to the cache. The directory updates/invalidates other caches with that entry.
  • Remote data cache: To reduce the overall data being transmitted
  • Enables: Up to 32 CPU sockets in a single machine
